# my-test-markup-plugin README

Тестовый плагин для упрощения работы в данном редакторе кода. 
Реализовал некоторый функционал для собственного удобства: подстветка/выделения выбранного участка текста и поиск информации определнного функционала языка программирования C++ в cppreference.com.

Авторство: Кущёв Данила Дмитриевич, группа: M3110, ИСУ: 407995
## Features

В данном расширении реализовал следующий функционал:

* Поиск информации в cppreference.com по выделенному тексту (т.е. если нужно быстро получить какую-либо информацию по различным библиотекам или встроенным функциям (и тому подобное), то будет достаточно выделить интересующее ключевое слово и вызвать команду плагина).

* Выделение выбранного текста.


> Tip: плагин-расширение писался исключительно в целях упрощения работы для автора. А также для выполнения лабораторной работы №3 и №3.1 по дистициплине "Инструментальные средства разработки ПО".
